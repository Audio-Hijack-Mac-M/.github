## Overview

Audio Hijack Advanced is a powerful audio recording and routing application from Rogue Amoeba that captures audio from any application, input device, or internet stream using a visual block-based session interface that makes complex audio routing both intuitive and flexible.

The block-based session design represents audio sources, effects, and outputs as connected visual elements, making it possible to build complex audio routing and processing setups — mixing multiple sources, applying effects, recording and streaming simultaneously — in a clear, understandable layout.

Application audio capture intercepts the audio output of any running application independently, enabling recording of music players, video conferencing tools, browser audio, and any other application without affecting other audio destinations.

Broadcast and live stream blocks enable direct RTMP streaming to platforms without requiring a separate streaming application, making Audio Hijack Advanced a complete audio solution for online broadcasters and live streamers.

For podcasters, radio producers, online broadcasters, and anyone who needs to capture, process, and route audio from multiple sources simultaneously on a Mac, Audio Hijack Advanced is the recording environment that handles every audio workflow with the quality and flexibility that Rogue Amoeba is known for.

## Performance

Audio Hijack balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
